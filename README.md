## 🎨 페이지 기능

### 1. 홈 페이지
- 히어로 섹션 (메인 비주얼)
- 기능 소개 (지그재그 레이아웃)
- 진행 중인 펀딩 카드
- 서비스 통계 (애니메이션 카운터)
- 아티스트 이미지 슬라이더 (그라디언트 카드)
- 파트너 로고 슬라이더 (브랜드 컬러 카드)
- **통일된 GNB**: 홈 / 조공하기 / 제휴사 / 문의하기

### 2. 펀딩 생성 프로세스 (4단계)
- **Step 1**: 아티스트 및 목표 설정
- **Step 2**: 카테고리 선택 (광고/식사/선물/굿즈)
- **Step 3**: 상세 옵션 선택
- **AI 분석 로딩**: 스크롤되는 4단계 분석 화면
  - 각 단계별 팬덕이 이미지와 분석 내용 표시
  - 자동 스크롤 애니메이션
  - 진행률 표시 바
  - 체크리스트 애니메이션
- **Step 4**: AI 추천 결과 확인

### 3. AI 예산 할당
- 랜덤 가중치 기반 예산 분배
- 카테고리별 맞춤 추천
- **비율 재조정 기능**
  - 주사위 애니메이션과 함께 비율 재계산
  - 카드 페이드인 애니메이션으로 변화 강조
  - 자동 스크롤로 결과 확인 용이

### 4. 제휴사 페이지 (신규)
- **카테고리별 분류**: 광고 / 식사&간식 / 선물 / 팬 굿즈 4개 분야별 파트너 소개
- **시각적 디자인**:
  - 브랜드 컬러를 활용한 입체적인 로고 카드 (Round + Shadow)
  - 마우스 오버 시 확대 및 그림자 효과 (Hover Animation)
  - 스크롤 시 순차적으로 나타나는 페이드인 효과
- **연동**: 각 파트너사 카드 클릭 시 공식 홈페이지 이동 (현재는 데모 링크 적용)

### 5. 사용자 경험 (UX)
- **상태 초기화**: 조공하기 시작 시 이전 데이터 자동 리셋
- **뒤로가기 기능**: Step 4에서 Step 3로 돌아가 옵션 재선택 가능
- **시각적 피드백**: 모든 주요 액션에 애니메이션 효과
